# 🤖 Rule-Based AI Chatbot

A simple rule-based chatbot built in Python that responds to predefined user inputs using conditional statements (`if-elif-else`). This project demonstrates the fundamentals of conversational AI, control flow, and decision-making logic.

## 📌 Project Overview

This chatbot simulates a basic conversation by matching user inputs against predefined rules and returning appropriate responses. It runs continuously until the user enters an exit command.

## 🚀 Features

* Handles greetings such as:

  * Hello
  * Hi
  * Hey

* Responds to common questions:

  * How are you?
  * What is your name?

* Handles exit commands:

  * Bye
  * Exit
  * Quit

* Uses:

  * `if-elif-else` decision-making logic
  * Continuous loop (`while True`)
  * Input normalization with lowercase conversion

## 🛠 Technologies Used

1. Install Python 3.

3. Open a terminal in the project directory.

```bash

```
